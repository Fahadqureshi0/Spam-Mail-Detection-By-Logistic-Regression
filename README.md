# 📧 Spam Mail Detection by Logistic Regression

A Machine Learning project that classifies emails as **Spam** or **Ham (Not Spam)** using the **Logistic Regression** algorithm.

## 📌 Project Overview

This project builds a binary classification model to detect spam emails based on their text content. The email messages are preprocessed, converted into numerical features using text vectorization, and classified using **Logistic Regression**.

## 🚀 Features

- Data preprocessing and cleaning
- Text preprocessing
- Feature extraction using TF-IDF Vectorization
- Model training using Logistic Regression
- Model evaluation
- Predictive system for classifying new emails

## 🛠️ Technologies Used

- **Language:** Python
- **Environment:** Google Colab
- **Libraries:**
  - NumPy
  - Pandas
  - Scikit-learn

## 🤖 Machine Learning Details

- **Problem Type:** Supervised Learning
- **Task:** Binary Classification
- **Algorithm:** Logistic Regression

## ▶️ How to Run

1. Clone this repository.
2. Open the notebook in **Google Colab** or **Jupyter Notebook**.
3. Install the required Python libraries.
4. Run all notebook cells.
5. Enter a new email message to predict whether it is **Spam** or **Ham**.
